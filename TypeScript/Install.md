# **[TypeScript](./Index.md) - Установка и обновление**

## Установка с помощью npm

Последняя стабильная версия:

```bash
npm install -g typescript
```

Ночная сборка:

```bash
npm install -g typescript@next
```

Обновление:

```bash
npm update -g typescript
```

## Команды

Проверить текущую версию:

```bash
tsc -v
```

Компиляция файла:

```bash
tsc app.ts
```
